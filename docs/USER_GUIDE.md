# Guide utilisateur Omed Scripture

## 1. Présentation

Omed Scripture est une application web de lecture biblique. Elle permet de lire des chapitres, changer de traduction, comparer des versions, écouter le texte, et garder des repères personnels (favoris, notes, surlignages).

L'application propose aussi des parcours de lecture et une synchronisation avec Google Drive pour conserver vos données entre plusieurs appareils.

## 2. Lire un chapitre

1. Ouvrez l'onglet **Lire**.
2. Choisissez une traduction dans la première liste.
3. Choisissez le livre biblique dans la deuxième liste.
4. Choisissez le chapitre dans la troisième liste.
5. Utilisez **Chapitre précédent** et **Chapitre suivant** pour avancer rapidement.

## 3. Changer de traduction

Depuis l'écran de lecture :

1. Ouvrez la liste de traductions en haut de page.
2. Sélectionnez la version souhaitée.
3. Le chapitre courant est rechargé avec la nouvelle traduction.

Remarque : selon la traduction, la source de texte n'est pas la même, ce qui peut entraîner des différences de style ou de disponibilité.

## 4. Comparer deux traductions

1. Dans le lecteur, cliquez sur **Comparer**.
2. Une deuxième colonne s'affiche à droite.
3. Choisissez la traduction de comparaison dans la liste de cette colonne.
4. Lisez les deux versions côte à côte, sur le même livre et le même chapitre.

## 5. Écouter l’audio

1. Dans le lecteur, cliquez sur **Audio**.
2. Le lecteur audio s'ouvre en bas de l'écran.
3. Utilisez **Lecture/Pause** pour contrôler l'écoute.
4. Le bouton de vitesse (ex. `1.0x`, `1.25x`) modifie la vitesse pour les chapitres suivants.

Important : l'audio repose sur la synthèse vocale du navigateur. La qualité, la voix et certaines options peuvent varier selon votre appareil et votre navigateur.

## 6. Sauvegarder un passage

Pour enregistrer un verset en favori :

1. Dans un chapitre, cliquez sur un verset.
2. Dans le menu d'actions, cliquez sur l'icône **cœur**.
3. Le verset est ajouté à vos **Favoris**.

Pour le retirer, répétez l'action sur le même verset.

## 7. Ajouter une note

1. Cliquez sur un verset pour ouvrir ses actions.
2. Cliquez sur l'icône **note**.
3. Saisissez votre texte dans la fenêtre qui s'affiche.
4. Validez pour enregistrer.

## 8. Surligner un verset

1. Cliquez sur un verset.
2. Cliquez sur l'icône de **surlignage**.
3. Choisissez une couleur.
4. Le surlignage est appliqué immédiatement.

Pour retirer le surlignage, sélectionnez à nouveau la même couleur.

## 9. Utiliser les parcours de lecture

1. Ouvrez **Plans de lecture** dans la navigation.
2. Choisissez un plan (par exemple 30, 90 ou 365 jours).
3. Commencez le plan puis ouvrez son détail.
4. Marquez les jours réalisés pour suivre votre progression.

## 10. Synchroniser avec Google Drive

1. Ouvrez **Paramètres**.
2. Connectez-vous avec votre compte Google si nécessaire.
3. Dans la section de synchronisation, lancez une synchronisation.
4. Activez la synchronisation pour conserver vos données (favoris, notes, surlignages, progression, position de lecture).

Si vous utilisez plusieurs appareils, faites une synchronisation après des modifications importantes.

## 11. Limites actuelles

- La lecture audio utilise la voix du navigateur, et non un enregistrement humain.
- La vitesse audio affichée s'applique surtout au prochain chapitre.
- Certaines traductions peuvent dépendre de services externes, avec des écarts de disponibilité.
- Le rendu et les performances peuvent varier selon le navigateur et l'appareil.

## 12. Résolution de problèmes courants

### Je ne vois pas mes données sur un autre appareil

- Vérifiez que vous êtes connecté au même compte Google.
- Lancez une synchronisation manuelle depuis **Paramètres**.
- Vérifiez l'état de synchronisation affiché dans l'application.

### L'audio ne démarre pas

- Vérifiez que le son de l'appareil est actif.
- Testez un autre navigateur récent.
- Rechargez le chapitre puis relancez l'audio.

### Une traduction ne se charge pas

- Vérifiez votre connexion Internet.
- Réessayez quelques instants plus tard (service externe possiblement indisponible).
- Testez une autre traduction pour confirmer si le problème est global ou limité à une version.

### Mes actions ne semblent pas enregistrées

- Rechargez la page pour vérifier la persistance locale.
- Si la synchronisation est activée, relancez une synchronisation depuis **Paramètres**.
